ABOUT THIS SITE

This website was built by the Bridgewater Raritan High School Technology Students Association (BRHS TSA) for the high school Webmaster competition. This site was created to promote pianist Trinity Fan and demonstrate the impact music had on her life.

This site was built using the Bootstrap framework and the programming languages HTML and CSS. In using this framework, the website is compatible with numerous common technological devices.

All images were either taken by BRHS TSA members or royalty free images from Icons8, Pixabay, and Pexels. Course descriptions were provided by the BRHS Program of Studies.
